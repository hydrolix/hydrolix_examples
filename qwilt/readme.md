# Export Qwilt logs to Hydrolix

Use the configuration files in this repo to export Qwilt HTTP transaction logs to Hydrolix and generate a Grafana dashboard to view the exported data.

## Before you begin

Make sure you have the following set up prior to exporting Qwilt logs to Hydrolix

* [A running Hydrolix cluster](https://docs.hydrolix.io/docs/welcome)
* [A Qwilt Cloud account](https://docs.qwilt.com/docs/login-to-qc-services-1)

## Installation

See the [Hydrolix documentation for ingesting Qwilt logs](https://docs.hydrolix.io/docs/qwilt-logs) for setup details.
